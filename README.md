# Debug Tools

This is a library for the debugging tools which I generally use in my Laravel Applications

Feel free to fork and share your quick-scripts that you use for development purposes

## WARNING

These tools are strictly to be used in development environment and under no conditions should it be in production environment. There may be some tools which may exploit your data publically and your data security might be compromised.

## Tools in the library

1. Quick Database search
2. Formatters (JSON, XML, unserialize, Base64 Encoder/Decoder)
3. Linked Multi-Search
4. Database Heavy Query Logger
